# Cat Breed Helper

Веб-приложение для определения породы кошки по фотографии.

Сейчас проект работает с мок-ответом, поэтому интерфейс и пользовательский сценарий уже готовы без подключения реальной ML-модели. Позже можно подключить настоящий backend endpoint без переработки фронтенда.

## Возможности

- загрузка изображения кошки через интерфейс
- предпросмотр выбранного изображения
- удаление выбранного изображения
- отображение результата распознавания
- вывод предполагаемой породы
- вывод уровня уверенности
- отображение источника ответа
- мок-режим для локальной разработки
- возможность дальнейшего подключения реального API

## Стек

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Flask или FastAPI
- мок-ответ на этапе разработки
- в дальнейшем подключение реального ML endpoint

## Как это работает

1. Пользователь выбирает изображение кошки.
2. Приложение показывает предпросмотр.
3. После отправки изображения фронтенд получает ответ от backend.
4. Пока вместо модели можно использовать мок-ответ.
5. Позже backend можно переключить на настоящий ML-сервис.
